Top Keyboard Shortcuts for Microsoft Word

Although you're familiar with Microsoft Word, you might be surprised by the multitude and diversity of keyboard shortcuts you can utilize to expedite your work and enhance convenience.

Now, is anyone expecting you to commit all these keyboard combinations to memory? Certainly not! Everyone's requirements differ, so some shortcuts will prove more beneficial to you than others. Yet, even acquiring a handful of new techniques is worthwhile. We've endeavored to present a concise and straightforward list, so feel free to print it for your reference!

Moreover, despite the considerable length of our shortcut compilation here, it's not an exhaustive inventory of every keyboard combo accessible in Word. We've focused on the most generally useful shortcuts. Furthermore, you'll be pleased to learn that nearly all of these shortcuts have stood the test of time, remaining useful regardless of the Word version you're using.

Essential Program Key Combinations

Microsoft Word offers numerous convenient keyboard shortcuts for various tasks, such as saving your document or reversing an action. These shortcuts streamline the use of the program and enhance efficiency.

Key Combination Description
Ctrl+N Create a new document
Ctrl+O Open an existing document
Ctrl+S Save the current document
F12 Open the Save As dialog box
Ctrl+W Close the current document
Ctrl+Z Undo the last action
Ctrl+Y Redo the last undone action
Alt+Ctrl+S Split a window or remove the split view
Ctrl+Alt+V Switch to Print Layout View
Ctrl+Alt+O Switch to Outline View
Ctrl+Alt+N Switch to Draft View
Ctrl+F2 Open Print Preview
F1 Open the Help pane
Ctrl+F Search within the document
F7 Run a spelling and grammar check
Shift+F7 Open the thesaurus; if a word is selected, it looks up that word in the thesaurus

Navigating within a document

You can employ keyboard shortcuts for efficient document navigation. This is particularly useful for long documents, allowing you to swiftly move between sections without extensive scrolling.

Keyboard Shortcut Description
Left/Right Arrow Shift the insertion point (cursor) one character to the left or right.
Ctrl+Left/Right Arrow Jump one word to the left or right.
Up/Down Arrow Navigate up or down one line.
Ctrl+Up/Down Arrow Move up or down one paragraph.
End Directly move to the end of the current line.
Ctrl+End Immediately jump to the end of the document.
Home Swiftly relocate to the beginning of the current line.
Ctrl+Home Instantly jump to the start of the document.
Page Up/Page Down Scroll up or down one screen at a time.
Ctrl+Page Up/Page Down Navigate to the previous or next browse object (after performing a search).
Alt+Ctrl+Page Up/Page Down Jump to the top or bottom of the current window.
F5 Access the Find dialog box with the "Go To" tab selected, allowing for quick navigation to specific pages, sections, or bookmarks.
Shift+F5 Toggle through the last three insertion point locations. When opening a document, Shift+F5 returns you to your last editing point before closing the file.

Efficient Text Selection Methods

From the previous section, you learned that arrow keys are for navigating the insertion point, and the Ctrl key changes how you move. Using the Shift key with these combinations enables diverse text selection techniques.

Keyboard Shortcut Description
Shift+Left/Right Arrow Expand your current selection one character to the left or right.
Ctrl+Shift+Left/Right Arrow Expand your current selection one word to the left or right.
Shift+Up/Down Arrow Extend the selection up or down by one line.
Ctrl+Shift+Up/Down Arrow Extend the selection to the beginning or end of the paragraph.
Shift+End Extend the selection to the end of the current line.
Shift+Home Extend the selection to the beginning of the current line.
Ctrl+Shift+Home/End Extend the selection to the beginning or end of the document.
Shift+Page Down/Page Up Extend the selection down or up by one screen.
Ctrl+A Select the entire document.
F8 Enter selection mode. Use arrow keys to expand the selection:
1st press: Enter selection mode
2nd press: Selects the word next to the insertion point
3rd press: Selects the entire sentence
4th press: Selects all characters in the paragraph
5th press: Selects the entire document
Press Shift+F8 to cycle backward. Press Esc to exit selection mode.
Ctrl+Shift+F8 Select a column. Use left/right arrow keys to extend the selection across columns.

Reviewing and Editing

Word also offers several keyboard shortcuts designed for text editing.

Key Combination Description
Backspace Delete one character to the left
Ctrl+Backspace Delete one word to the left
Delete Delete one character to the right
Ctrl+Delete Delete one word to the right
Ctrl+C Copy text or graphics to the Clipboard
Ctrl+X Cut selected text or graphics to the Clipboard
Ctrl+V Paste the Clipboard contents
Ctrl+F3 Cut selected text to the Spike. The Spike is an interesting variant on the regular clipboard. You can keep cutting text to the Spike and Word remembers it all. When you paste the Spike's contents, Word pastes everything you cut, but places each item on its own line.
Ctrl+Shift+F3 Paste the Spike contents
Alt+Shift+R Copy the header or footer used in the previous section of the document

Implementing character formatting

Word is equipped with numerous keyboard combinations that allow you to apply character formatting. These shortcuts can be used either on selected text or on text that you type next when no text is currently selected.

Key Combination Description
Ctrl+B Apply bold formatting
Ctrl+I Apply italic formatting
Ctrl+U Apply underline formatting
Ctrl+Shift+W Apply underline formatting to words, but not the spaces between words
Ctrl+Shift+D Apply double underline formatting
Ctrl+D Open the Font dialog box
Ctrl+Shift+< or > Decrease or increase font size one preset size at a time
Ctrl+[ or ] Decrease or increase font size one point at a time
Ctrl+= Apply subscript formatting
Ctrl+Shift+Plus key Apply superscript formatting
Shift+F3 Cycle through case formats for your text. Available formats are sentence case, lowercase, uppercase, title case, and toggle case.
Ctrl+Shift+A Formats all letters as uppercase
Ctrl+Shift+K Formats all letters as lowercase
Ctrl+Shift+C Copies the character formatting of a selection
Ctrl+Shift+V Pastes formatting onto selected text
Ctrl+Space Removes all manual character formatting from a selection

Applying formatting to paragraphs

Similar to character formatting, Word offers a variety of shortcuts specifically for formatting paragraphs

Key Combination Description
Ctrl+M Increase a paragraph's indent one level each time you press it
Ctrl+Shift+M Reduce a paragraph's indent one level each time you press it
Ctrl+T Increase a hanging indent each time you press it
Ctrl+Shift+T Reduce a hanging indent each time you press it
Ctrl+E Center a paragraph
Ctrl+L Left-align a paragraph
Ctrl+R Right-align a paragraph
Ctrl+J Justify a paragraph
Ctrl+1 Set single-spacing
Ctrl+2 Set double-spacing
Ctrl+5 Set 1.5 line spacing
Ctrl+0 Remove one line spacing preceding a paragraph
Ctrl+Shift+S Open a popup window for applying styles
Ctrl+Shift+N Apply the normal paragraph style
Alt+Ctrl+1 Apply the Heading 1 style
Alt+Ctrl+2 Apply the Heading 2 style
Alt+Ctrl+3 Apply the Heading 3 style
Ctrl+Shift+L Apply the List style
Ctrl+Q Remove all paragraph formatting

Adding content

Whether you want to insert a section break in your document or quickly add a common symbol without searching, Word's keyboard shortcuts provide convenient solutions.

Key Combination Description
Shift+Enter Insert a line break
Ctrl+Enter Insert a page break
Ctrl+Shift+Enter Insert a column break
Ctrl+- (hyphen) Insert an optional hyphen or en dash
Alt+Ctrl+- (hyphen) Insert an em dash
Ctrl+Shift+- (hyphen) Insert a non-breaking hyphen
Ctrl+Shift+Spacebar Insert a non-breaking space
Alt+Ctrl+C Insert a copyright symbol
Alt+Ctrl+R Insert a registered trademark symbol
Alt+Ctrl+T Insert a trademark symbol

Managing outlines

For those who prefer to outline before diving into a lengthy document, here are some handy shortcuts to assist you.

Key Combination Description
Alt+Shift+Left/Right Arrow Promote (move to the left) or demote (move to the right) a line
Ctrl+Shift+N Demote an outline level to regular body text
Alt+Shift+Up/Down Arrow Move the line with the insertion point up or down in the outline
Alt+Shift+Plus or Minus keys Expand or collapse text under a heading
Alt+Shift+A Expand or collapse all text or headings in an outline
Alt+Shift+L Show the first line of body text or all body text
Alt+Shift+1 Show all headings that have the Heading 1 style applied
Alt+Shift+any other number key Show all headings up to that level

Manipulating tables

Navigating within tables requires a different approach compared to regular text. Instead of clicking to move around, utilize these keyboard shortcuts:

Key Combination Description
Tab Move to the next cell in a row and select its contents, if there are any
Shift+Tab Move to the previous cell in a row and select its contents, if there are any
Alt+Home/End Move to the first or last cell in a row
Alt+Page Up/Page Down Move to the first or last cell in a column
Up/Down Arrow Move to the previous or next row
Shift+Up/Down Arrow Select the cell in the row above or below the insertion point or selection. Keep pressing this combo to keep selecting more cells. If you have multiple cells in a row selected, this combo selects those same cells in the row above or below.
Alt+5 on keypad (with NumLock off) Select an entire table
