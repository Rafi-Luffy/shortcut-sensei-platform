Top 79 Shortcuts for Slack

Welcome to Shortcut Sensei!
Are you a frequent Slack user looking to optimize your productivity? You've come to the right place! Slack is packed with shortcuts that can help you navigate channels, manage messages, and collaborate with ease.


Discover the Possibilities
In this guide, we've compiled the top 75 Slack shortcuts, sorted into categories for easy reference. Whether you're sending messages, managing notifications, or customizing your workspace, these shortcuts will supercharge your Slack experience. Let's dive in and explore how these shortcuts can revolutionize the way you work in Slack!


General Shortcuts

Master the basics of Slack with these essential shortcuts for navigation, communication, and workspace management.

Key Combination Description
Ctrl + Shift + K Open direct messages
Ctrl + Shift + L Open channels list
Ctrl + Shift + T Create a new channel
Ctrl + , (Comma) Open preferences
Ctrl + Q Quit Slack
Ctrl + Period (.) Open the keyboard shortcuts list

Navigation Shortcuts

Effortlessly move through Slack with these shortcuts, helping you quickly access channels, direct messages, and other key areas.

Key Combination Description
Ctrl + K Open the Quick Switcher
Ctrl + 1 to 9 Switch to a specific workspace (based on the order in the sidebar)
Ctrl + Tab Move to the next section/detail pane
Ctrl + Shift + Tab Move to the previous section/detail pane
Ctrl + ] Move to the next channel or DM
Ctrl + [ Move to the previous channel or DM
Ctrl + Shift + J Move to next section in the sidebar
Ctrl + Shift + K Move to previous section in the sidebar
Ctrl + ` (Grave accent) Toggle between your direct messages and channels
Ctrl + Shift + U Jump to unread messages
Page Up Scroll up in the current channel/views
Page Down Scroll down in the current channel/views
Home Jump to the beginning of the conversation
End Jump to the most recent message

Messaging Shortcuts

Enhance your communication with these shortcuts for composing, editing, and managing your messages in Slack.

Key Combination Description
Ctrl + Enter Send a message
Shift + Enter Start a new line in the message
Ctrl + Shift + * (Asterisk) Insert a line break without sending the message (similar to Shift + Enter)
Ctrl + B Bold selected text
Ctrl + I Italicize selected text
Ctrl + Shift + X Strikethrough selected text
Ctrl + Shift + 7 Format selected text as a numbered list
Ctrl + Shift + 8 Format selected text as a bulleted list
Ctrl + Shift + > (Greater Than) Quote selected text
Ctrl + Shift + B Format selected text as code block
Ctrl + L Hyperlink selected text
Ctrl + Shift + M Open the Mention window

Search and Filters Shortcuts

Quickly find messages, files, and more using these shortcuts for efficient searching and filtering in Slack.

Key Combination Description
Ctrl + F Open the search bar for current conversation
Ctrl + G Open the search bar to search across all conversations
Ctrl + Shift + F Open the search bar and pre-select Messages tab
Ctrl + Shift + G Open the search bar and pre-select Files tab

Channels and Mentions Shortcuts

Manage your channels and stay on top of mentions with these handy shortcuts for better team communication.

Key Combination Description
Ctrl + Shift + M View recent mentions
Ctrl + Shift + A View the Activity pane
Ctrl + Shift + S Deactivate notifications for a channel or direct message

Files and Attachments Shortcuts

Streamline file management with these shortcuts for uploading, accessing, and organizing files and attachments in Slack.

Key Combination Description
Ctrl + Shift + Y Open the files pane
Ctrl + U Upload a file

Calls and Video Shortcuts

Efficiently manage your audio and video calls with these shortcuts, making it easy to connect and communicate in Slack.

Key Combination Description
Ctrl + Shift + H Start a call
Ctrl + Shift + H again End the call
Ctrl + Shift + A Toggle the video on/off
Ctrl + M Mute/unmute the microphone

Emojis and Reactions Shortcuts

Express yourself quickly with these shortcuts for adding emojis and reactions to messages in Slack.

Key Combination Description
Ctrl + Shift + * (Asterisk) Open the emoji menu
Ctrl + Shift + E Add a reaction to the last message

Interface Shortcuts

Navigate and customize the Slack interface effortlessly with these essential shortcuts.

Key Combination Description
Ctrl + = (Equal) Zoom In
Ctrl + - (Minus) Zoom Out
Ctrl + 0 Reset Zoom
Ctrl + \ (Backslash) Toggle sidebar visibility
F11 Toggle full-screen mode

Direct Messages Shortcuts

Stay on top of your direct message conversations with these shortcuts for efficient communication in Slack.

Key Combination Description
Ctrl + Shift + D Open direct messages with a specific user
Ctrl + Shift + Up Arrow Go to newer messages in direct messages
Ctrl + Shift + Down Arrow Go to older messages in direct messages

Threads Shortcuts

Manage and participate in message threads with ease using these shortcuts for threaded conversations in Slack.

Key Combination Description
Ctrl + T Open threads view
Ctrl + Shift + . (Period) Jump to most recent thread

Notifications Shortcuts

Take control of your Slack notifications with these shortcuts for managing alerts and staying informed.

Key Combination Description
Ctrl + Shift + M Open the Mentions & Reactions pane
Ctrl + Shift + N Next unread channel or direct message
Ctrl + Shift + K Open All Unreads

Administration Shortcuts

Streamline administrative tasks and manage your Slack workspace efficiently with these shortcuts.

Key Combination Description
Ctrl + Shift + C View channel details
Ctrl + / (Forward Slash) Open the Commands list
Ctrl + Shift + A Open the Accessibility settings
Ctrl + Y Set or clear your status
Ctrl + Shift + I Open developer tools
Ctrl + Shift + F Format code (pre-set formatting)

Keyboard Focus Shortcuts

Quickly navigate and interact with Slack using these shortcuts for keyboard-focused actions.

Key Combination Description
Esc Remove focus from the text input
Ctrl + Shift + 1 Focus on the first workspace
Ctrl + Shift + 2 Focus on the second workspace, and so on
Alt + Shift + Up Arrow Move focus up in the sidebar
Alt + Shift + Down Arrow Move focus down in the sidebar

Common Actions Shortcuts

Perform common actions swiftly with these shortcuts, enhancing your productivity in Slack.

Key Combination Description
Ctrl + Shift + P Create a new workspace
Ctrl + Shift + R Reload Slack
Alt + Shift + 0-9 Navigate through your organization's different workspaces
Ctrl + Shift + C Mark all messages as read

Miscellaneous Shortcuts

Explore a variety of handy shortcuts for miscellaneous tasks in Slack, improving your overall user experience.

Key Combination Description
Alt + Left Arrow Go back to previous view
Alt + Right Arrow Go forward to next view
Ctrl + E Go to recent conversations
Ctrl + Shift + / (Forward Slash) Open search for shortcuts and commands
